# colony_analysis
 

## requirement

#### mac os
```shell
brew cask install r
```

#### ubuntu
```shell
apt install r-base
```

## install colonylive
```shell
pip install git+https://github.com/takeshi30rou/colonylive_v2
```
or
```shell
git clone https://github.com/takeshi30rou/colonylive_v2.git
cd colonylive_v2
pip install .
```
